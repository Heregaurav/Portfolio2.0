# Backend for Coding Profile API

This folder contains the standalone backend for aggregating coding platform profiles.

## Installation

Copy `.env.example` to `.env` and fill values.

```bash
cd backend
npm install
```

## Run

```bash
npm run dev
# or
npm start
```

API endpoints are mounted under `/api` (e.g. `GET /api/github`).

See `src` for implementation.
