# Portfolio Backend

Aggregates coding profiles into a single API for a personal portfolio.

## Tech
- Node.js
- Express
- Axios
- GraphQL Request
- dotenv
- node-cache
- CORS
- Helmet
- Morgan

## Installation

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies:

```bash
npm install
```

3. Run backend:

```bash
npm run dev:backend
```

## Docker

```bash
docker-compose up --build
```
